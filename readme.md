Simple Alias for adding an AI as a co-author to your commit.

Currently supported:
- ChatGPT
- Claude
- Cursor

All you need to do is add `alias gitca='/yourpath/Gitca/gitca.sh` to your PATH and voala!.

## How to use

run `gitca claude` and it will amended to the last commit from the current directory with the given ai that you have chosen.
